# 🌦️ WeatherNetwork

A modern iOS Weather Application built using **SwiftUI and MVVM Architecture** that provides real-time weather information for any city using weather API integration.



 📱 Overview

WeatherNetwork allows users to search for cities and view current weather conditions in a clean, modern, and interactive UI.

The application demonstrates:

* SwiftUI UI Development
* MVVM Architecture
* Networking & API Integration
* State Management
* ObservableObject & Published Properties
* Dynamic UI Updates
* Light / Dark Mode Support
* Reusable Components
* Animations



 ✨ Features

✅ Search weather by city name

✅ Real-time weather updates

✅ Current temperature display

✅ Weather description

✅ Humidity information

✅ Feels Like temperature

✅ Wind Speed

✅ Animated weather card

✅ Modern Glassmorphism UI

✅ Dynamic Gradient Background

✅ Git & GitHub Integration

          ↓

          ↓

          ↓
